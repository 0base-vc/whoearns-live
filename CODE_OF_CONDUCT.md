# Code of Conduct

This project adopts the spirit of the Contributor Covenant (see
<https://www.contributor-covenant.org/>) without reproducing its text
verbatim. The short version below is the rule set we actually enforce.

## Expected behavior

- Treat other contributors with respect regardless of experience level,
  background, or identity.
- Give and receive feedback on code and ideas — not on people. Prefer
  concrete, actionable suggestions over vague judgments.
- Use inclusive language. Assume good faith when a message is ambiguous.
- Credit the work of others, and be generous about acknowledging where
  your patch builds on theirs.
- Keep discussions on-topic. Disagreement is fine; dragging unrelated
  conflicts into an issue or PR is not.

## Unacceptable behavior

- Harassment, discrimination, or personal attacks of any kind.
- Unwanted sexual attention, slurs, or imagery in any project space.
- Publishing someone else's private information without their consent.
- Sustained disruption of discussion, deliberate intimidation, or
  repeated off-topic derailment.

## Enforcement

If you see or experience any of the above, report it to
`conduct@whoearns.live`. Reports are reviewed confidentially by the project
maintainers. Depending on severity the response may be a private reminder, a
public clarification, a temporary ban from project spaces, or a permanent ban.

We will not retaliate against anyone for reporting in good faith. We
will decline to act on reports that are themselves bad-faith attempts
to weaponise this policy against another contributor.

## Scope

This code applies to all project spaces: the GitHub repository, issue
tracker, pull requests, discussions, any chat channel tied to the
project, and any event where someone is representing the project.
